# White Wine Quality Prediction

A regression analysis project that predicts white wine quality scores (0–10 scale) based on eleven physicochemical properties.

## Problem

Given a set of measurable chemical attributes of white wine, can we accurately predict its quality rating? This project applies multiple regression techniques to answer that question.

## Dataset

The dataset contains white wine samples with the following features:

- Fixed acidity
- Volatile acidity
- Citric acid
- Residual sugar
- Chlorides
- Free sulfur dioxide
- Total sulfur dioxide
- Density
- pH
- Sulphates
- Alcohol

**Target variable:** Quality score (0–10)

## Approach

Multiple regression techniques are compared and evaluated in `projecte_vi.Rmd`, with a full analysis report available in the PDF.
